# Linux Command Guides
Use these for daily Linux terminal operations.

## Start order (beginner)
1. File basics: `ls.md`, `cat.md`, `mkdir.md`, `mv.md`, `touch.md`.
2. Search/filter: `find.md`, `grep.md`, `sed.md`, `awk.md`, `xargs.md`.
3. System/network: `systemctl.md`, `journalctl.md`, `ssh.md`, `curl.md`.
4. Packaging/backup: `tar.md`, `rsync.md`.

## Safety rule
- For commands that can delete/overwrite, run the preview variant first.
