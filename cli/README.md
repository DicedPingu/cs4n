# CLI Guides
Use these for everyday terminal productivity and repository operations.

## Start order
1. `github.md` if you are learning pull/commit/push/PR flow.
2. `rg.md` for fast text search.
3. `tmux.md` for persistent terminal sessions.
4. `wget.md` for resilient downloads.

## Editor integration
- Open current repo in VS Code with `code .`.
- If you prefer the name `code-insiders`, alias it: `alias code-insiders='code'`.
